# Base Implementation - Reference Files

This directory contains the core implementation files from the original Roth project that the Home Assistant custom component is based on.

## Reference Files

### Core API Implementation
- **api_reference.py** - Raw HTTP implementation (from examples/raw_http_test.py)
- **complete_system_data.json** - Full system API documentation and data

### Original Project Files  
- **main_demo.py** - Main application demonstration
- **project_readme.md** - Original project documentation

These files serve as reference and testing material for the Home Assistant integration.
